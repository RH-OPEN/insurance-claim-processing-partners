== オリジナル アプリ

左側の最初の項目メニューである「オリジナル アプリ」では、保険会社で使用されていた古いアプリケーションがどのようなものかを確認できます。各請求をナビゲートすると、インターフェイスが非常に粗雑で、最初に受信したメールの単純なコピーと添付された写真だけが表示されていたことがわかります。

== プロトタイプ アプリ

左側の 2 番目の項目メニューである「新しいアプリ」では、新しいアプリケーションのプロトタイプがどのようなものかを確認できます。各請求をナビゲートすると、インターフェイスがはるかにユーザー フレンドリーで、請求をナビゲートして添付された写真を見るための優れたインターフェイスが備わっていることがわかります。

さらに、処理された請求については、この請求の概要、顧客感情分析、請求から抽出されたイベントの場所と時間、AI によって処理された写真を確認できます。

[.bordershadow]
image::05/new-app-1.png[]

これは明らかにまだ非常にシンプルで粗雑なアプリケーションですが、これはプロトタイプだったことを忘れないでください。そして、AI 強化アプリケーションの可能性を取締役会に示し、プロジェクトへの投資を説得するには十分でした。
